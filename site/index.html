JetKit/CDK

An anti-framework for building serverless TypeScript applications on AWS. A library designed to make writing serverless functions and RESTful APIs as lightweight and convenient as possible.

NPM   GitHub

By building on AWS, API Gateway, Lambda, and CDK it is possible to focus on implementing application logic and cloud infrastructure.

You don't need to waste time managing services or containers.
Deployments, monitoring, alerting, routing, tracing, and serverless functions are all provided by AWS. Effortlessly utilize best practices by default.
